# Bert-text-classification
### 模型

基于Bert作为baselin模型，分别用BERT，BERT+CNN，BERT+RNN，BERT+RCNN，BERT+DPCNN，ERNIR，ERNIR+DPCNN模型来做分本分类

ERNIE：增强性BERT，和BERT一样的用法，主要用于中文



### 数据集

使用THUCNews做为数据集



### 结果分析

|    模型     | Test_Acc |
| :---------: | :------: |
|    BERT     |  88.54%  |
|  BERT+CNN   |  88.34%  |
|  BERT+RNN   |  87.42%  |
|  BERT+RCNN  |  88.01%  |
| BERT+DPCNN  |  88.40%  |
|    ERNIR    |  94.09%  |
| ERNIR+DPCNN |  94.20%  |
